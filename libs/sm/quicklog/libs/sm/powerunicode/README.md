# ðŸ¸ Description
A PowerShell Class for parsing Unicode names, codes, and characters from the Unicode.org website and outputting them in a JSON Codex file. This can then be parsed and use by other class functions.

## â†ª Class Properties

| **Name** | **Definition** |
| --- | --- |
| `[String]Name` | A string that represents the name of a Unicode character. |
| `[String]Unicode` | A string that represents the Unicode code of a character. |
| `[String]Label` | A string that represents a cleaned up version of the Unicode character name. |
| `[String]Codex` | An object that represents a collection of the parsed Unicode characters. |

| Name           	| Return 	| Description 	|
|--------------------	|----------	|-----------------	|

| `PrintByLabel()`     	|   `[string]`       	| Gets unicode by label.                	|
| `PrintByUnicode()`   	|   `[string]`      	| Gets unicode by Unicode.                	|
| `UnicodeGet()`       	|    `[void]`      	| Retrieves the Unicode information from the `unicode.org` website and filters the data to get the unicode names and codes. The information is then outputted as a JSON file in the script directory. Input: None.      	|
